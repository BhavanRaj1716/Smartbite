# 🍽 SmartBite - Smart Canteen

A real-time canteen ordering system built with HTML, CSS, JavaScript and Firebase.

## Features
- 🔐 Email & Phone (OTP) authentication
- 🛒 Real-time cart and order tracking
- 💳 UPI, Card and Cash payments
- 📱 Mobile UPI deep-link (auto-opens UPI app with exact amount)
- 🔒 Dynamic QR code with locked payment amount
- 👨‍💼 Admin panel with live order management

## Tech Stack
- HTML, CSS, JavaScript
- Firebase Authentication
- Firebase Firestore (real-time database)
- Firebase Hosting

## Setup
1. Clone the repo
2. Replace Firebase config in `script.js` and `admin.js` with your own
3. Update `UPI_ID` in `script.js` with your UPI ID
4. Update `ADMIN_EMAIL` in `admin.js` with your admin Google email
5. Deploy to Firebase Hosting
